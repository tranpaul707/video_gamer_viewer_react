# video_gamer_viewer_react

  This project serves as a basic web application to observe viewable video games. The main goal of this project is to first understand how to manage and create a react project, using available technolgies like Typescript, React, and other related tools.

This application in short summary, will allow users to filter games based on genres, search through thousands of games through a search tool, and have a nice UI to view and look at all of the hottest games available now.
